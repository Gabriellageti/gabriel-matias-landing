# Gabriel Matias Landing Page

Landing page pessoal/profissional para Gabriel Lage Pires Matias, com foco em monitoração, observabilidade, NOC, dashboards, automação e inteligência operacional.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## Como rodar localmente

```bash
npm install
npm run dev
```

Depois acesse a URL exibida pelo Vite, normalmente `http://localhost:5173`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Segurança de conteúdo

Esta página não deve expor informações internas da Energisa ou de qualquer cliente. Antes de adicionar imagens reais em `src/assets/dashboard`, remova ou substitua:

- nomes de hosts reais
- IPs
- URLs internas
- nomes de clientes
- credenciais
- e-mails internos
- incidentes reais
- qualquer dado operacional sensível

Quando houver dúvida, use mockups anonimizados.
